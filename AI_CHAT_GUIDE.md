# AI时间管理助手 - 使用指南

## 🤖 功能介绍

AI时间管理助手是一个智能对话系统，专门为时间管理和效率优化设计。它能够：

### 核心能力

1. **智能任务规划**
   - 根据自然语言描述生成结构化的任务列表
   - 自动识别任务优先级和时间安排
   - 提供合理的时间分配建议

2. **暗时间挖掘**
   - 分析你的日程，找出可以优化的时间段
   - 建议并行任务和时间复用策略
   - 识别通勤、等待等碎片时间的利用机会

3. **个性化建议**
   - 基于你的工作模式和身份特征
   - 推荐适合的时间管理方法（GTD、番茄钟、双峰工作法等）
   - 提供具体可执行的工具和方法

4. **深度洞察**
   - 分析你的效率模式和能量曲线
   - 识别时间盲区和效率瓶颈
   - 提供"顿悟时刻"式的优化建议

## 🚀 快速开始

### 第一步：配置AI API

1. 点击导航栏的 **"API配置"** 或右上角的设置图标
2. 选择AI服务提供商：
   - **OpenAI** - 推荐使用 GPT-3.5-turbo 或 GPT-4
   - **DeepSeek** - 国内服务，响应快速
   - **自定义** - 支持兼容OpenAI格式的其他服务

3. 填写配置信息：
   ```
   API提供商: OpenAI
   API密钥: sk-xxxxxxxxxxxxxxxx
   模型: gpt-3.5-turbo
   温度: 0.7 (推荐)
   最大Token: 2000
   ```

4. 点击 **"测试连接"** 验证配置
5. 点击 **"保存配置"** 完成设置

### 第二步：开始对话

1. 点击导航栏的 **"AI助手"** 进入聊天页面
2. 在输入框中描述你的需求
3. 按 Enter 发送（Shift+Enter 换行）
4. AI会实时回复并提供建议

#### 1. 详细描述你的情况

**❌ 不好的提问：**

告诉AI你的：

首次进入时，可以点击快捷问题快速开始：
- "帮我规划今天的工作任务"
- "如何提高工作效率？"
- "我的通勤时间很长，如何利用？"
- "推荐一些时间管理工具"

## 📋 使用场景示例

### 场景1：早晨规划

**你的输入：**
```
早上好！我今天有以下任务：
1. 写一份技术方案（重要且紧急）
2. Code Review 3个PR
3. 参加2个会议（10:00和15:00）
4. 学习新技术（想学但不紧急）

我是程序员，上午精力最好，下午容易困。
请帮我安排今天的时间。
```

**AI会提供：**
- 基于能量曲线的任务排序
- 会议间隙的时间利用建议
- 学习任务的碎片化方案
- 避免下午困倦的方法

### 场景2：暗时间挖掘

**你的输入：**
```
我每天通勤2小时（地铁），中午午休1小时，
晚上8-10点是家庭时间。这些时间段我该如何利用？
```

**AI会分析：**
- 通勤时间：听播客、有声书、思考规划
- 午休时间：NSDR休息法、轻度学习
- 家庭时间：高质量陪伴的同时处理简单任务

### 场景3：效率优化

**你的输入：**
```
我是内容创作者，经常写作时灵感枯竭，
而且容易被社交媒体分心。有什么好方法吗？
```

**AI会建议：**
- 工具推荐：副屏、大屏幕、专注软件
- 方法论：番茄钟、深度工作时段
- 灵感管理：随时记录、定期整理
- 环境优化：断网写作、专注空间

## ⚙️ 高级配置

### API配置参数说明

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| **Temperature** | 控制回复的创造性 | 0.7 |
| **Max Tokens** | 单次回复的最大长度 | 2000 |
| **Model** | AI模型选择 | gpt-3.5-turbo |

**Temperature 调整建议：**
- `0.3-0.5`: 更精确、一致的回复
- `0.7-0.8`: 平衡创造性和准确性（推荐）
- `0.9-1.0`: 更有创意但可能不够精确

### 支持的AI服务

#### OpenAI
```
API Endpoint: https://api.openai.com/v1/chat/completions
推荐模型: gpt-3.5-turbo, gpt-4
```

#### DeepSeek
```
API Endpoint: https://api.deepseek.com/v1/chat/completions
推荐模型: deepseek-chat
```

#### 自定义服务
任何兼容OpenAI格式的API服务都可以使用

## 🔒 隐私和安全

### 数据存储
- API密钥加密存储在浏览器本地
- 对话历史仅保存在当前会话
- 不会上传到我们的服务器

### API调用
- 直接从浏览器调用AI服务
- 不经过中间服务器
- 遵循各AI服务商的隐私政策

### 建议
- 不要在对话中包含敏感个人信息
- 定期更换API密钥
- 使用子账号或受限权限的API密钥

## 🐛 常见问题

### Q: 为什么AI没有回复？

**可能原因：**
1. API密钥未配置或已过期
2. API额度不足
3. 网络连接问题
4. API服务暂时不可用

**解决方法：**
1. 检查API配置页面
2. 使用"测试连接"功能验证
3. 查看浏览器控制台的错误信息
4. 尝试更换API服务商

### Q: 如何获取API密钥？

**OpenAI:**
1. 访问 https://platform.openai.com
2. 注册/登录账号
3. 进入 API Keys 页面
4. 创建新的API密钥

**DeepSeek:**
1. 访问 https://platform.deepseek.com
2. 注册/登录账号
3. 进入API管理页面
4. 创建API密钥

### Q: API调用费用如何计算？

- **OpenAI**: 按Token计费，GPT-3.5约$0.002/1K tokens
- **DeepSeek**: 价格更低，具体查看官网
- **建议**: 设置月度预算限制

### Q: 对话历史会保存吗？

- 当前会话的对话会保存在浏览器内存中
- 刷新页面或关闭浏览器后会清空
- 可以点击"清空对话"手动清除
